# Digit Entry Views

A full-screen view prompting users to enter a series of digits (e.g., a PIN) using a digit-specific keyboard. **tvOS only** — not supported on iOS, iPadOS, macOS, visionOS, or watchOS.

An optional title and prompt can be displayed above the line of digits.

## Best Practices

- **Use secure digit fields.** Secure digit fields display asterisks instead of the entered digit. Always use secure fields when asking for sensitive data.
- **Clearly state the purpose.** Use a title and prompt that explains why someone needs to enter digits.
